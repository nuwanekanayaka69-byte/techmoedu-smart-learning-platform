# Chapter 05 – System Architecture

## Overview

The TechmoEDU Smart Learning Platform follows a modern cloud-based web architecture.

The system is designed using React for the frontend, Firebase Authentication for secure login, Firestore as the cloud database, Firebase Storage for file storage, and Firebase Hosting (or Vercel) for deployment.

This architecture provides scalability, security, maintainability, and high availability.
