# 04 - Backend Architecture

## Overview

The backend of the TechmoEDU Smart Learning Platform will be powered by Firebase services.

## Backend Technology

- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Firebase Hosting
- Firebase Security Rules

## Authentication

Firebase Authentication will manage:

- User login
- User logout
- Password reset
- Role-based access

## Database

Firestore will store:

- Users
- Institutes
- Teachers
- Students
- Parents
- Courses
- Classes
- Attendance
- Marks
- Fees
- Payments
- Notices
- Resources

## Storage

Firebase Storage will manage:

- Student profile images
- Teacher profile images
- PDF notes
- Gallery images
- Uploaded resources

## Security

Security will be controlled using:

- Authentication
- Role-based permissions
- Firestore Security Rules
- Storage Security Rules
