# 02 - Activity Diagram

This folder contains UML Activity Diagram documentation for the TechmoEDU Smart Learning Platform.

## Purpose

Activity Diagrams describe the workflow of important system processes.

They show:

- Start point
- User actions
- System actions
- Decisions
- Alternative paths
- End point
