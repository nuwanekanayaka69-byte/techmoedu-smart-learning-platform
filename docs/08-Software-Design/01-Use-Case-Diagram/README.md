# 01 - Use Case Diagram

## Overview

This section documents the Use Case Model of the TechmoEDU Smart Learning Platform.

The Use Case Diagram illustrates how different actors interact with the system and what services are available to each user role.

---

## Objectives

- Identify system actors
- Define system functionality
- Illustrate interactions between actors and the system
- Support software analysis and design

---

## Primary Actors

- Super Administrator
- Institute Administrator
- Teacher
- Student
- Parent
