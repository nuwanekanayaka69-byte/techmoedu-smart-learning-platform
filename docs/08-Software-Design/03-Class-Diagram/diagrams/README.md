# Class Diagram Files

This folder contains all UML Class Diagram files.

---

## Diagram Formats

Each diagram is stored in three formats.

| Format | Purpose |
|---------|----------|
| Draw.io | Editable source |
| PNG | Documentation preview |
| PlantUML | Source code |

---

## Folder Structure

```
diagrams/
├── drawio/
├── png/
└── plantuml/
```
